# Nama Aplikasi
Aplikasi SIAKAD (Sistem Informasi Akademik) berbasis web yang dibangun
menggunakan Laravel untuk mengelola data akademik mahasiswa.

## Halaman & Fungsi

## Halaman Login
Terdapat 2 role: Admin dan Mahasiswa

## Halaman Admin

### Dashboard
Menampilkan ringkasan informasi akademik mahasiswa seperti total dosen, mahasiswa, mata kuliah, jadwal, dan KRS terbaru

### Dosen
Mengelola data dosen

### Mahasiswa
Mengelola data mahasiswa

### Mata Kuliah
Mengelola daftar mata kuliah

### Jadwal
Mengelola data jadwal

### Kartu Rencana Studi (KRS)
Mengelola data KRS dan export ke pdf

## Halaman Mahasiswa

### KRS Mahasiswa
Mengambil mata kuliah dan export ke pdf

### Jadwal Kuliah
Menampilkan jadwal kuliah
